# Contributing to ClimbIndia

Thanks for helping build the Indian climbing community!  
Here’s how to contribute:

1. **Add a new crag**
   - Create a file under `/data/<state>/<crag-name>.json`
   - Use the format in `data/maharashtra/dukes_nose.json`
2. **Add safety or access notes**
   - Create or edit markdown files under `/docs/`
3. **Suggest improvements**
   - Open an Issue in GitHub (e.g., “Add crags near Pune”)

### Pull request checklist
- JSON file validates (you can test via https://jsonlint.com)
- Latitude/longitude are correct
- No personal names or private land info
